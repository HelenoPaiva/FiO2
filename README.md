# FiO₂ Calculator
https://helenopaiva.github.io/FiO2/

Minimal, mobile-first FiO₂ calculator for anesthesia use.

Designed for fast use in the OR: instant results, no buttons, clean UI, and works offline as a single HTML file.

---

## Features

- Real-time FiO₂ calculation from:
  - Air flow (L/min)
  - O₂ flow (L/min)

- Reverse calculation:
  - Input total flow + target FiO₂
  - Get required Air and O₂ flows

- Mobile optimized:
  - iOS numeric keypad
  - Handles decimal comma and dot (e.g. `1,5` or `1.5`)
  - No clicks required — updates as you type

- Fully offline:
  - Single `.html` file
  - No dependencies
